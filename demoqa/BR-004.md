# BR-004 — Web Tables: поле Age принимает 00/0 и помечает значение как валидное

- **Severity:** Medium
- **Priority:** Medium
- **Окружение:** Windows 11 Pro, Google Chrome (143.0.7499.193)
- **Страница:** [Elements → Web Tables](https://demoqa.com/webtables)

## Preconditions
1. Открыта страница Web Tables

## Steps to Reproduce
1. Нажать кнопку **Add**
2. В поле **Age** ввести `00`
3. Кликнуть вне поля (проверить отображение валидации)

## Actual Result
Поле **Age** принимает значение `00` (после потери фокуса может отображаться как `0`) и помечается как валидное (зелёная рамка и галочка).

## Expected Result
Age не должен принимать 0 (значение `00` должно считаться невалидным). Поле должно подсвечиваться ошибкой и не считаться валидным.
